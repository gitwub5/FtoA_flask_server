+ 프로젝트는 가상환경에서 진행되였습니다.
+ **virtualenv 가상환경 생성**
    
    **가상환경 설치 & 생성**
    
    ```
    $ sudo pip3 install virtualenv
    ```
    
    virtualenv를 설치했다면 프로젝트 폴더에서 가상환경을 생성하고 켜주자!
    
    ```
    $ virtualenv myenv
    $ source myenv/bin/activate
    ```
